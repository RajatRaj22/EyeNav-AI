# 👁️ EyeNav AI

### *Hands-Free Computer Control using Eye Movements & Blink Intelligence*

---

## 🚀 Overview

**EyeNav AI** is a real-time computer vision system that enables users to control a computer **without using hands**, relying entirely on **eye movements and blink patterns**.

Built as an assistive technology solution, the system transforms a standard webcam into a powerful human–computer interaction tool, allowing users to **navigate, scroll, and even type using Morse code via eye blinks**.

---

## 🎯 Key Highlights

✔ Real-time eye tracking using facial landmark detection
✔ Blink-based Morse code typing system
✔ Gaze-controlled navigation (scrolling, page switching)
✔ Fully offline — no cloud or internet dependency
✔ Lightweight and optimized for standard laptops
✔ Designed for accessibility and assistive use cases

---

## 🧠 How It Works

```text
Webcam Input
     ↓
Face Detection (MediaPipe)
     ↓
Eye Landmark Extraction
     ↓
Blink Detection + Gaze Tracking
     ↓
Morse Code Interpretation / Navigation Commands
     ↓
System Control (Scroll, Type, Navigate)
```
